50 terms for web development.
1. html - defines an HTML document 2. head - defines information about the document 3. title - defines a title for the document 4. body - defines the document's body 5. h1 to h6 - defines HTML headings
6. p - defines a paragraph 7. br - inserts a single line break 8. hr - defines a thematic change in the content 9. !--...-- - defines a comment 10. a - defines a hyperlink
11. link - defines the relationship between a document and an external resource 12. nav - defines navigation links 13. ul- defines an unordered list 14. ol - defines an ordered list 15. li - defines a list item
16. table - defines a table 17. tr - defines a row in a table 18. td - defines a cell in a table 19. style - defines style inforatmion for a document 20. div - defines a section in a document
21. span - defines a section in a document 22. header - defines a header for a document 23. footer - defines a footer for a document 24. main - specifies the main content of a document 25. section - defines a section in a document
26. head - defines information about the document 27. meta - defines metadata about an HTML document 28. align - specifies the alignment according to surrounding elements 29. border - specifies the width of the border of an element 30. href - specifies the URL of the page the link goes to
31. video - defines a video or a movie 32. margin - creates space around elements 33. padding - defines the innermost potuon of the box model 34. hover - used to select elements when you mouse over them 35. active - represents an element which is activated by user
36. min-height - defines the minimum height of an element 37. max-height - defines maximum height of an element 38. font-family - specifies the font for an element 39. font-size - specifies the size of font 40. background-color - sets the background color of an element
41. display - specifies the display behavior of an element 42. float - specifies that an element should be placed along the left or right side of its container 43. color - defines the color of an element 44. list-style - defines how a list is displayed 45. list-style-type - specifies the type of list-item
46. height - specifies the content height of boxes 47. width - specifies the width of an element 48. background-position(x,y) - sets the starting position of a background image 49. flex - the flex property sets the flexible length on flexible items 50. position - specifies the type of positioning method used for an element
